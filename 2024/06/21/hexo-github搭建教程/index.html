安装下载部署hexo

创建文件夹（命名随便）
右键文件夹选git bash打开
切换国内路线 npm config set registry https://npmmirror.com

npm install -g hexo-cli安装hexo
hexo init初始化· 没这一步文件夹内的文件显示不出来


出现这些文件才算成功，否则有问题
要么自行百度要么问(我的微信15913105505)

然后

输入hexo g和hexo s
就会出现以下界面(恭喜你本地部署成功)
